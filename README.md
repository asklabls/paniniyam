# Paniniyam

A static web app for studying Pāṇini's grammar — browse the Ashtadhyayi and related texts in any Indic script.

**Live site → [asklabls.github.io/paniniyam](https://asklabls.github.io/paniniyam/)**

---

## What's inside

**Texts**
- Ashtadhyayi — all 3,976 sūtras with padaccheda, anuvṛtti, and adhikāra
- Kāśikā Vṛtti and Laghu Kaumudī commentary per sūtra
- Dhātupāṭha — 2,259 roots across 10 gaṇas with conjugation tables
- Gaṇapāṭha, Uṇādi Kośa, Liṅgānuśāsanam, Śikṣā, Fiṭ Sūtras, Śiva Sūtras
- Tiṅ-pratyaya reference tables (adanta and anadanta, sārvadhatuka and ārdhadhātuka)

**Features**
- Read in 11 scripts — Telugu, Kannada, Malayalam, Tamil, Bengali, Gujarati, Gurmukhi, Odia, Devanagari, IAST, ITRANS
- Audio pronunciation for every sūtra
- Search across all texts
- Fully static — no login, no server, no tracking
