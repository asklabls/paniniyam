This tool uses the Sanscript.js library to convert Sanskrit text between scripts. Text converts automatically as you type. Supported schemes:

Indic scripts (Devanagari, Telugu, Tamil, etc.) are writing systems native to South Asia. Romanization schemes (IAST, ITRANS, HK, SLP1, Velthuis) use Latin letters — each with different conventions — to represent Sanskrit sounds in ASCII or Unicode.

IAST is the scholarly standard for publications. ITRANS is common in online forums and Devanagari input. Harvard-Kyoto (HK) and SLP1 are used in computational linguistics. Velthuis is an older TeX convention.
